"Eu prometo um resultado!"

"Produzir código" é um código que pode levar algum tempo

“Consumindo código” é o código que deve aguardar o resultado

Uma promessa é um objeto que vincula a produção de código e o consumo de código

# Propriedades do objeto promessa

let minhaPromessa = new Promise(function(minhaResolucao, minhaRejeicao) { // "Código que produz" (Pode levar algum tempo) minhaResolucao(); // quando bem sucedido minhaRejeicao(); // quando erro }); // "Código que consome" (Deve esperar por uma promessa cumprida) minhaPromessa.then( function(value) { /* código se for bem sucedido */ }, function(error) { /* código se algum erro */ } );

Quando o código produtor obtiver o resultado, ele deverá chamar um dos dois callbacks:

Quando Chama
Sucesso minhaResolucao()
Erro minhaRejeicao()
- O objeto Promise oferece suporte a duas propriedades: estado e resultado. - Enquanto um objeto Promise está como "pendente" (funcionando), o resultado é indefinido. - Quando um objeto Promise é "cumprido", o resultado é um valor. - Quando um objeto Promise é "rejeitado", o resultado é um objeto de erro.
